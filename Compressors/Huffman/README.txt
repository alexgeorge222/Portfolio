Huffman Compression

This program takes in a user text input, and displays the Huffman tree used to compress it, and the decompresses the text using a 
code bank created during the decompression process

Required Files:
BinaryNode.java
BinaryNodeInterface.java
CompressDecompress.java*
CompressDecompressGUI.java(main program)
HuffmanTree.java
MinHeap.java
MinHeapInterface.java
